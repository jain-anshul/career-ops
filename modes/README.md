# Modes Reference

| Mode | Summary |
|------|---------|
| `auto-pipeline` | Full end-to-end flow when you paste a JD or URL — evaluates, writes report, generates PDF, updates tracker |
| `oferta` | Scores a single job offer A–F across 7 dimensions with a structured report |
| `ofertas` | Side-by-side comparison and ranking of multiple offers |
| `pdf` | Generates an ATS-optimized, JD-tailored CV as HTML+PDF |
| `pipeline` | Processes a batch of pending URLs from `data/pipeline.md` |
| `batch` | Runs parallel workers to evaluate multiple offers at once |
| `scan` | Scrapes configured job portals for new offers matching your target roles |
| `apply` | Live application assistant — fills forms and drafts answers, stops before submitting |
| `contacto` | Finds the right LinkedIn contact at a company and drafts an outreach message |
| `deep` | Deep-dives a company's strategy, culture, risks, and interview signals |
| `tracker` | Displays current application statuses, pipeline health, and action items |
| `patterns` | Analyzes rejection patterns across all applications to improve targeting |
| `followup` | Flags overdue follow-ups and generates follow-up message drafts |
| `interview-prep` | Produces interview process intel — round structure, reported questions, story bank mapping |
| `interview` | Builds a fine-grained ML/AI study plan with prioritized topics and curated resources |
| `training` | Evaluates a course or certification against your career North Star |
| `project` | Evaluates a portfolio project idea for career ROI |
| `_shared` | Shared context (profile, archetypes, scoring weights) loaded by most modes |
| `_profile` | Your personal customizations — archetypes, narrative, comp targets, deal-breakers |
